# Projects

## 1. Banking SQL Project

### Project Description

A complete banking database system built using relational database principles.
The project includes normalized tables, relationships, analytical queries, stored functions, views, and triggers.

### Database Structure

The system contains interconnected tables such as:

* Customers
* Accounts
* Branches
* Loans
* Transactions

### Concepts Used

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* GROUP BY & HAVING
* Aggregate Functions
* Subqueries
* Views
* Stored Functions
* Triggers
* Date Functions
* Normalization

### Business Problems Solved

* Customer account summary
* Branch-wise account analysis
* Loan exposure analysis
* Dormant account detection
* Monthly transaction trends
* Multi-loan customer identification
* Customer 360° analysis

### Example Insights

* Top customers by loan amount
* Branches with low activity
* Customers with multiple active loans
* Monthly transaction growth trends
